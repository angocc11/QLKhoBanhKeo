# Menu Icons

This folder contains icons for the menu buttons in the application.

## Required Icons

For the best visual appearance, please add the following icons to this folder:
- partners.png - For "Quản lý đối tác" button
- transaction.png - For "Quản lý giao dịch" button
- statistics.png - For "Thống kê" button
- products.png - For "Quản lý hàng hóa" button
- inventory.png - For "Kiểm kê kho" button
- user-add.png - For "Tạo tài khoản" button
- logout.png - For "Đăng xuất" button

## Icon Specifications

- Size: 24x24 pixels is recommended
- Format: PNG with transparency
- Color: White or light color to show well on the blue background

## Where to Find Icons

You can download free icons from:
1. https://www.flaticon.com
2. https://icons8.com
3. https://material.io/resources/icons

After downloading, place them in this folder and rename them according to the list above.
The application will automatically load them when it starts.
