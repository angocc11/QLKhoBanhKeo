# Resources Folder

This folder is for storing application resources like images and other assets.

## How to add the store logo:

1. Add a PNG file named "store.png" to this folder to display the store logo
2. The application will automatically load the image when it starts

## Store logo specifications:

- Format: PNG with transparency
- Recommended size: 64x64 pixels
- If no image is provided, a default logo with "BK" text will be generated
